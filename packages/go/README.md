# spikard - Go Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

```bash
go get github.com/Goldziher/spikard
```

## Quick Start

```go
package main

import "github.com/Goldziher/spikard"

func main() {
	// See https://github.com/kreuzberg-dev/spikard for usage examples.
}
```
