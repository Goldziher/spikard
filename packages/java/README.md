# spikard - Java Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

Add to your `pom.xml`:

```xml
<dependency>
    <groupId>dev.spikard</groupId>
    <artifactId>spikard</artifactId>
    <version>0.14.0</version>
</dependency>
```

## Quick Start

```java
import dev.spikard.*;

// See https://github.com/kreuzberg-dev/spikard for usage examples.
```
