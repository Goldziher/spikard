# spikard - Dart Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

Add to `pubspec.yaml`:

```yaml
dependencies:
  spikard:
    git: <repo-url>
```

## Quick Start

```dart
// Phase 2: Dart bindings via flutter_rust_bridge. Skeleton only.
```
