# spikard - Elixir Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

Add `:spikard` to your `mix.exs` dependencies:

```elixir
defp deps do
  [
    {:spikard, "~> 0.14.0"}
  ]
end
```
