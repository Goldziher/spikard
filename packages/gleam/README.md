# spikard - Gleam Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

```sh
gleam add spikard
```

## Quick Start

```gleam
import spikard

// Call functions exported by the generated module.
// The NIF is loaded via `@external(erlang, "Elixir.Spikard.Native", ...)`.
```
