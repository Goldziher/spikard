# spikard - Kotlin Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

Add the generated package to your `build.gradle.kts`:

```kotlin
dependencies {
    implementation("dev.spikard:spikard:VERSION")
}
```

## Quick Start

```kotlin
import dev.spikard.Spikard

// Call generated APIs through the Spikard object.
```
