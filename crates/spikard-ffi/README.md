# spikard - FFI (C/C++) Bindings

Rust-centric multi-language HTTP framework with polyglot bindings

## Installation

Link against `libspikard_ffi` and include `spikard.h`.

See the build instructions in the main repository.

## Quick Start

```c
#include "spikard.h"

int main(void) {
    // See https://github.com/Goldziher/spikard for usage examples.
    return 0;
}
```
