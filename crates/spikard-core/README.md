# spikard-core

Shared transport-agnostic primitives for Spikard runtimes — route, router, method, schema validator. Re-exported by [`spikard`](https://crates.io/crates/spikard) and consumed by every binding crate.

For framework documentation and usage examples, see the [Spikard repository](https://github.com/Goldziher/spikard).
