# Benchmark Visualization Charts

This directory contains auto-generated HTML charts from the latest benchmark runs.

## Files

- `throughput-by-framework.html` - Request throughput comparison across frameworks
- `latency-percentiles.html` - Latency distribution (P50, P90, P95, P99)
- `validation-overhead.html` - Validation vs raw performance comparison
- `resources.html` - CPU and memory utilization
- `metadata.json` - Benchmark run metadata
- `aggregated.json` - Complete aggregated benchmark results

## Generation

Charts are automatically generated by the `visualize-benchmarks` GitHub Actions workflow after
the `comparative-benchmarks` workflow completes successfully.

### Manual Generation

To generate charts locally:

```bash
# Download and aggregate latest benchmark results
task benchmarks:aggregate

# Generate visualization charts
task benchmarks:visualize
```

Or using the CLI directly:

```bash
# Aggregate results from latest successful run
./target/release/benchmark-harness aggregate

# Generate charts
./target/release/benchmark-harness visualize \
  --input benchmark-results/aggregated.json \
  --output docs/assets/benchmarks
```

## Notes

- These files are committed to git for immediate availability in `mkdocs serve`
- They are regenerated automatically in CI when benchmarks complete
- Charts use Plotly for interactive visualizations
- Theme colors match the MkDocs Material design
